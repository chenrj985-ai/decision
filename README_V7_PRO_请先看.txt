V7 Pro Windows人工盘面增强版
================================

本版是在V6腾讯行情主源完整工程上直接升级的可运行程序，不是功能说明。

一、第一次使用
1. 解压整个文件夹。
2. 双击 INSTALL_V7_PRO.cmd。
3. 编辑 data\my_positions.csv。
4. 双击 EDIT_HUMAN_VIEW.cmd，录入今天的重点/回避板块和个股人工加减分。
5. 双击 RUN_V7_PRO.cmd。
6. 查看 output\mobile_latest.html。

二、主要改进
1. 市场画像六维：
   赚钱效应、市场情绪、资金承接、板块扩散、热点持续、追高安全。
2. 推荐树：
   A级可小仓、B级观察、C级等待回踩、D级禁追。
3. 不再因为危险指数低就强行凑推荐数量。
4. 人工盘面编辑器：
   重点板块、回避板块、允许追高、允许超跌试仓、
   板块加减分、个股加减分、盘面备注。
5. 板块生命周期：
   启动、发酵、高潮、分歧、退潮、震荡。
6. 每只股票显示：
   推荐理由、扣分理由、人工调整、交易教练分、最终动作。
7. 人工判断不能突破重大利空、爆雷指数、行业禁买等硬风控。

三、为什么危险指数31仍可能只有少数A级
危险指数回答的是“市场整体是否危险”；
A级数量回答的是“有多少股票同时满足行业强、位置不过热、
资金有承接、相对强、爆雷低、没有追高”等条件。
市场安全但没有好买点时，A级可以很少，但B/C观察池会更丰富。

四、人工使用建议
人工加分建议控制在：
+5：略看好
+8：重点关注
+12：盘面确认很强
-8：不喜欢
-15：明确退潮
-25：今日完全回避

五、Windows兼容
启动脚本会依次寻找：
系统Python、用户Anaconda、D盘Anaconda、ProgramData Anaconda。
CMD使用UTF-8和CRLF，减少中文乱码与闪退。
